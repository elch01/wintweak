# Windows powershell tweaks

Bunch of powershell scripts

## Disclaimer
Remember these need an unrestricted execution policy so type "set-executionpolicy unrestricted" before running them, dont forget to set the policy back to default with "set-executionpolicy restricted"

**I am not responsive for any damage to your computer do this at your own risk!**

## Contents

### thispc

Removes or adds the contents of This PC and only leaves Drives and Removable storage

### misc

Small tweaks, may later be incorporated into a single file for all commands.
